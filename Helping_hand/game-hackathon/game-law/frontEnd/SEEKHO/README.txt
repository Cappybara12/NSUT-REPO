# Aao Seekhein 

Aao seekhein is an interactive educational website made for illiterate children and their parents and aims to promote english education in rural areas through interactive games and videos. The website is made entirely in hindi to increase readability.

# Steps to open locally

• Clone the project from github

• Install an extension such as live server

• Open index.html in aaoseekhein/frontend/SEEKHO

• Click on run with live server

• The website opens up
